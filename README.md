# rock-paper-scissors
First JavaScript project following The Odin Project curriculum.

For this project we are creating our own rock paper scissors game. First step will be planning and pseudocoding the solution.

 The game will be played vs the computer, first I need to get the computer choice randomly. Wether is rock, paper or scissors. 

1. Create a function named getComputerChoice that return randomly "rock", "paper" or "scissors".

2. Write the logic to get the human choice. getHumanChoice. 
 - Get a number between 1 and 3 using prompt method.
 - If user's input is 1 that will return rock, 2 for paper and 3 for scissors.

 3. Keep track of players score in two variables humanScore and computerScore both initiating in 0.

 4. Write the logic to play a round using a function playRound with two parameters. humanChoice and computerChoice.
